# Itemly UI 设计提示词文档

> 本文档为 Itemly 物品管理系统的前端 UI 设计规范与提示词，可直接用于指导 AI 生成代码或向设计师传达视觉方向。

---

## 一、设计定位

**产品名称：** Itemly — 轻量化物品管理系统  
**目标用户：** 注重生活整理的个人用户，追求清晰、平静的视觉体验  
**核心感受：** 像一本精心整理的纸质账册——有条理、温暖、触感真实  
**设计风格：** 现代简约 + 有机温感（Modern Minimal with Organic Warmth）

---

## 二、色彩系统

| 角色 | 名称 | HEX |
|------|------|-----|
| 主色（背景深色模式 / 导航） | 深紫黑 | `#1E0A3C` |
| 画布背景 | 浅紫白 | `#F5F3FF` |
| 强调色（主按钮、激活态） | 电紫 | `#7C3AED` |
| 辅助色（标签、图标） | 电光青 | `#06B6D4` |
| 卡片背景 | 纯白 | `#FFFFFF` |
| 边框 / 分割线 | 紫灰 | `#EDE9FF` |
| 正文 | 深紫黑 | `#1A0E2E` |
| 次级文字 | 紫灰 | `#7C6F96` |
| 危险操作 | 玫红 | `#F43F5E` |

**使用原则：**
- 背景以 `#F7F5F0` 为主，营造纸张质感
- 琥珀金 `#D4A853` 仅用于主要行动按钮、高亮标记，保持视觉权重
- 鼠尾草绿 `#8BAF8E` 用于分类标签、成功状态等次级强调
- 卡片左侧使用彩色竖边框区分物品分类（每个分类对应一个柔和色调）

## 三、字体系统

```
显示字体（Logo、大标题）：Plus Jakarta Sans — 700/800
UI 字体（导航、按钮、标签）：Plus Jakarta Sans — 400/500/600
数据字体（数量、属性值）：Inter — 400/500
```

**字号比例（基于 16px）：**

| 层级 | 用途 | 大小 | 字重 |
|------|------|------|------|
| Display | 页面主标题 | 28px | 700 |
| Heading | 卡片标题、模块标题 | 18px | 600 |
| Body | 正文、描述 | 14px | 400 |
| Caption | 属性标签、时间戳 | 12px | 400 |
| Label | 按钮、导航项 | 13px | 500 |

## 四、布局结构

### 桌面端（≥1024px）
```
┌──────────────────────────────────────────────────────┐
│  顶部导航栏（Logo + 搜索框 + 用户头像）  64px 高      │
├──────────┬───────────────────────────────────────────┤
│          │  ┌──────────────────────────────────────┐ │
│  左侧    │  │  统计卡片区（物品数 / 分类数 / 属性数）│ │
│  侧边栏  │  └──────────────────────────────────────┘ │
│  240px   │  ┌──────────────────────────────────────┐ │
│          │  │  筛选条件展示栏                        │ │
│  · 导航  │  └──────────────────────────────────────┘ │
│  · 分类  │  ┌──────────────────────────────────────┐ │
│  · 属性  │  │                                      │ │
│  筛选树  │  │  物品卡片网格（3列）                   │ │
│          │  │                                      │ │
│          │  └──────────────────────────────────────┘ │
└──────────┴───────────────────────────────────────────┘
```

### 平板端（768px–1023px）
- 侧边栏收起为抽屉式（Drawer），顶部增加 ≡ 菜单按钮
- 卡片网格切换为 2 列

### 移动端（<768px）
- 顶部导航保留搜索 + 菜单按钮
- 卡片网格切换为 1 列
- 底部固定悬浮添加按钮（FAB）
- 分类/属性筛选通过底部抽屉打开

## 五、核心组件规范

### 5.1 物品卡片（Item Card）
```
┌──┬────────────────────────────────┐
│  │  [物品图片 — 4:3 比例]          │
│彩│                                │
│色│  物品名称 (Heading 600)         │
│边│  备注描述 (Body 灰色 两行截断)   │
│框│  ┌──────┐ ┌──────┐ ┌──────┐   │
│  │  │分类  │ │属性  │ │属性  │   │
│  │  └──────┘ └──────┘ └──────┘   │
│  │                   ⋮（更多）    │
└──┴────────────────────────────────┘
```
- 卡片圆角：12px
- 阴影：`0 2px 12px rgba(0,0,0,0.06)`
- Hover：上浮 `translateY(-3px)` + 阴影加深
- 左侧彩色竖边框：4px 宽，颜色根据分类映射

### 5.2 标签（Tag / Badge）
- 分类标签：圆角 999px，背景鼠尾草绿 20% 透明度，文字鼠尾草绿
- 属性标签：圆角 6px，背景 `#F0EDE8`，文字炭灰
- 可点击标签：hover 时背景加深，cursor: pointer

### 5.3 侧边栏
- 背景：`#1A2236`（深海军蓝）
- 文字：白色 / 60% 白色（次级）
- 激活项：琥珀金左侧指示线 3px + 文字变白
- 分类/属性树形展开：缩进 16px/层

### 5.4 顶部导航
- 背景：`#FFFFFF`，底部 1px `#E8E4DC` 边框
- 搜索框：圆角 8px，`#F7F5F0` 背景，聚焦时 `#D4A853` 边框
- 用户头像：32px 圆形

### 5.5 主按钮（Primary Button）
- 背景：`#D4A853`，文字白色，圆角 8px
- Hover：亮度提升 `brightness(1.08)`
- 加载态：内嵌 spinner

### 5.6 统计卡片
- 3 个横排卡片（桌面端），各含大数字 + 说明文字 + 趋势小图标
- 背景白色，圆角 12px，左上角彩色圆点装饰

### 5.7 悬浮添加按钮（FAB）
- 仅移动端显示
- 右下角固定，圆形 56px，琥珀金背景，白色 + 号
- 点击展开添加物品表单（底部抽屉）

## 六、页面列表

| 页面 | 路径 | 功能概述 |

| 首页/物品列表 | `/` | 卡片网格 + 搜索 + 筛选 |

| 统计面板 | `/stats` | 数量统计 + 分类分布图 |
| 设置 | `/settings` | 账号信息修改 |

## 七、微交互 & 动效

- 卡片进入动画：`fadeInUp` 错开 50ms，仅首次加载执行
- 侧边栏收展：`300ms ease-in-out` 宽度过渡
- 标签点击筛选：选中状态 `100ms` 背景色切换
- 图片放大：`scale(1) → scale(1)` 配合遮罩淡入，点击遮罩关闭
- 删除确认：卡片轻微左滑后消失，不使用弹窗打断（或使用 inline 确认）
- Toast 提示：右上角滑入，2.5s 后自动消失

---

## 八、辅助功能（Accessibility）

- 所有交互元素具备 `focus-visible` 轮廓（琥珀金 2px outline）
- 颜色对比度符合 WCAG AA 标准
- 分类颜色标识不作为唯一区分维度，搭配文字标签
- 支持 `prefers-reduced-motion`，关闭所有非必要动画

## 九、AI 代码生成提示词（可直接使用）

```
请为一个名为 Itemly 的个人物品管理系统设计并实现前端 UI。

设计要求：
- 风格：现代简约 + 有机温感，避免过度扁平化
- 色彩：主背景 #F7F5F0（暖白），导航背景 #1A2236（深海军蓝），强调色 #D4A853（琥珀金），辅助色 #8BAF8E（鼠尾草绿）
- 字体：Plus Jakarta Sans（标题/UI）+ Inter（数据/正文），均通过 Google Fonts 引入
- 布局：固定顶部导航 + 左侧侧边栏（桌面端 240px，移动端抽屉式）+ 右侧内容区
- 响应式：桌面 3 列卡片网格 → 平板 2 列 → 移动 1 列 + FAB 悬浮按钮
- 物品卡片：4:3 图片区域 + 标题 + 描述 + 分类/属性标签，左侧 4px 彩色分类指示边框
- 侧边栏：深色背景，支持分类树形展开，激活项显示琥珀金左边框
- 微交互：卡片 hover 上浮效果，标签点击高亮，页面首次加载卡片错开淡入
- 图片放大：点击卡片图片弹出遮罩层全屏预览，点击关闭
- 统计面板：顶部 3 个统计卡片（物品总数/分类数/属性数）
- 提供完整的 HTML + CSS + JavaScript 单文件实现，使用原生技术，无需框架
```
